to_s(format=nil)

Return a string-representation of the entry. If no formatter is given as argument, it first falls back to the formatter in @format (see Entry#format=) and if that is not present, to Entry::DefaultFormat Entry#to_s calls format on the formatter, passing it self as the first and only argument. See Log::Formatter for an example implementation.
